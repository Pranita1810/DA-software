Quick Summary : This is an Exponential Data Analytical project which focuses on extracting meaningful insights from data which helps
                to build business strategies and decision making which impacts business in positive way.
		This is a system designed data pipeline which sends reports each week and builds business strategy based on data with 
		the help of LLM.


Tech Stack    : Power BI, Python, SQL, LLM, ML, AI.
